Confused with align,justify etc? - not surprising!

simple div with another centered

Here we are using the default orientation thus the layout is 'row' so the main axis is horizontal and the cross axis is the vertical one.

align-items: center; on the container aligns the item VERTICALLY (on this orientation) - it actually aligns the item on the cross axis.

justify-content: center; on the container aligns the item HORIZONTALLY (on this orientation) - it actually aligns the item on the main axis.
